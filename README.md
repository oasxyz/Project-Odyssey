# Project-Odyssey
## A long-term self-learning journey to become a Fullstack Website Developer.

## What is Project Odyssey?
Project Odyssey is my personal software engineering journey.
Instead of following random tutorials, this repository documents a fundamentals learning through real projects, guided challanges, and continous reflection.
The goal isn't to finish courses.
The goal is to become a developer capable of understanding, designing, building, and maintaning real software.

## Why?
Like many beginners, I found myself jumping between tutorials, courses, YouTube videos, and documentation without a clear direction. (might be I am skill issues).
So I made Project Odyssey for me to grind, practice, and doing some real projects.
Or like many people called it "Practice by Doing".

## Philosophy
My philosophy is simple. (but with some notes, that I'd still reading documentations around the Internet).
Project > Tutorial
Understanding > Memorization
